# BB • Interactive Creative Hub

An aesthetic, premium, glassmorphic dashboard showcasing interactive design, creative scripting, and custom styling.

## 🚀 Live Demo
Visit the live application here:
**[https://suz41.github.io/BB/](https://suz41.github.io/BB/)**

## ✨ Features
- **Ambient Floating Orbs**: Smooth, multi-layered CSS background glow elements with natural floating animations.
- **Interactive Canvas Pad**: Responsive canvas that spawns dynamic stardust particles trailing your mouse pointer.
- **Ambient Soundscape**: An audio-context synthesizer that produces soft, cybernetic hum tones, paired with a synchronized mock visualizer.
- **Palette Playground**: Dynamic generation of tailored HSL color schemes that can be copied with a single click.
- **Personalization Suite**: Sidebar controls to modify background atmosphere themes, adjust card transparency/glass intensity, and toggle animations.
- **Ephemeral Notes**: LocalStorage-persisted workspace to write and store quick notes locally on your device.

## 🛠 Tech Stack
- **HTML5**: Semantic layout with interactive elements.
- **CSS3**: Modern custom properties (variables), backdrop-filters, custom grid layouts, and advanced keyframe animations.
- **JavaScript (Vanilla)**: Object-oriented canvas particles, AudioContext API, localStorage bindings, and real-time DOM management.
