# 🎫 O' Romeo - Mobile Ticket Dashboard

A premium, interactive, and fully responsive mobile ticket dashboard built from scratch using modern web standards. Recreated exactly from the design specifications of `t.jpg` and enhanced with a modern aesthetic, micro-animations, and full theme personalization.

## 🚀 Live Application
Visit the live deployment here:
👉 **[https://suz41.github.io/BB/](https://suz41.github.io/BB/)**

---

## ✨ Features
* 🌓 **Dynamic Theme System:** Seamless toggle between a warm light mode matching the original print specifications and a modern sleek dark mode.
* 🔍 **SVG Vector QR Code:** High-fidelity scalable vector QR code.
* 🎬 **Cinematic Trailer Modal:** Play mock movie trailers directly inside the UI.
* 📋 **Smart Sharing System:** Interactive share and copy actions that write the booking details (`T9AFDXQ`) directly to your clipboard alongside a toast confirmation.
* ⚙️ **Action Drawer:** A mobile-first slide-up panel featuring ticket management actions like PDF downloading, email confirmation, and booking cancellation.

---

## 🛠 Tech Stack
* **HTML5:** Clean semantic tags, accessible forms, and native SVG graphics.
* **CSS3:** Custom properties (variables) for theme control, viewport grids, absolute center layouts, custom keyframes, and smooth transition easing.
* **JavaScript:** Vanilla DOM manipulation, clipboard write events, local storage integration for persistence, and responsive interactive modal handlers.

## 💻 Local Development
To run this project locally, start a static server in the root directory:
```bash
# Using Node/npx
npx http-server -p 8080
```
Open **[http://127.0.0.1:8080](http://127.0.0.1:8080)** in your browser.
